About CryptoDash

CryptoDash is a real-time cryptocurrency dashboard built for fintech startups — demonstrating live API integration, responsive UI, and modern JavaScript architecture.

It showcases the startup's ability to handle live data and deliver a polished, trust-building experience before users subscribe to premium services.

Features

Live market tracking
Interactive 7-day charts
Favorites persistence
Dark mode support
Mobile-first layout
USD → INR converter
